Reflexão e Aplicação Prática

Lista das Propriedades CSS utilizadas e para que servem:

Propriedade justify-content

Trata-se de uma propriedade que alinha os itens horizontalmente e aceita os seguintes valores:

  • Flex-start: Os itens são alinhados ao lado esquerdo do contêiner;
  • Flex-end: Os itens são alinhados ao lado direito do contêiner;
  • Center: Os itens são alinhados no centro do contêiner;
  • Space-between: Os itens são exibidos com espaçamento igual entre eles;
  • Space-around: Os itens são exibidos com espaçamento igual ao redor deles;
  • Justify-content:flex-end : Moverá o item para a direita.

Aplicação Prática: Menu de Navegação

Para o Menu de Navegação, iremos dividi-lo em três itens, como: "início", "sobre" e "destaques". Tudo isso no topo da página de navegação. Nesse sentido, colocaremos os itens do menu em uma "div" e usaremos o "display:flex" para alinhar os mesmos. Além disso, utilizaremos o "align-items:center" para alinhá-los no centro e o "justify-content: space-around ou space-between" para que haja espaço distribuído entre eles.
